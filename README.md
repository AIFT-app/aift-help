# aift-help

Source for the AI Finance Team help center, hosted at https://help.aifinance.team.

Built with MkDocs Material. Articles live in `docs/` as markdown.

## Editing

- Local: clone, run `mkdocs serve`, edit, see live preview at http://127.0.0.1:8000
- GitHub web: navigate to any file in `docs/`, click the pencil icon, commit. Site rebuilds automatically in ~30 seconds.

## Adding a new article

1. Create `docs/<slug>.md`
2. Add an entry under `nav:` in `mkdocs.yml`
3. Commit and push to main
